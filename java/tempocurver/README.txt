To use this and all other Java (mxj) objects, install all *.class files in Max5/Cycling '74/java/classes.  Alternatively, install them in another directory and add the following line to Max5/Cycling '74/java/max.java.config.txt:

max.dynamic.class.dir /Users/john/myclasses

replacing /Users/john/myclasses with the path to your class folder.



Tempocurver was developed by Matthew Wright in collaboration with composer Edmund Campion.  The project was commissioned by the Drumming Ensemble of Portugal with Miquel Bernat, Director.  The piece, A Different Kind of Measure (ADKOM) was premiered in Porto, Portugal in 2001 and has since been performed by the Juilliard Percussion Ensemble and others.
